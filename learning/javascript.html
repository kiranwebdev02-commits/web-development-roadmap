JavaScript Programming Language

JavaScript is the programming language of the web. It makes websites interactive and dynamic.

⚡ JavaScript = Electricity of House

HTML builds the House. CSS paints the House. JavaScript gives Electricity. 💡 Bulb ON/OFF 🚪 Door Bell 🪟 Automatic Door 📺 TV Remote Everything starts working.

Basic JavaScript


document.getElementById("demo").innerHTML="Hello World";

This JavaScript code changes the text of an HTML element.
It makes the webpage interactive without reloading the page.

Important Points

  • JavaScript is the programming language of the web.
  • JavaScript makes websites interactive.
  • JavaScript responds to user actions like clicks and typing.
  • JavaScript can change HTML and CSS dynamically.
  • JavaScript is used for animations, forms and games.
  • JavaScript works together with HTML and CSS.
