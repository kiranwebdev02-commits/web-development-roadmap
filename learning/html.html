HTML (HyperText Markup Language)

HTML is the standard language used to create web pages. It provides the complete structure of a website.

🏠 HTML = Structure of House

Think of a house. Foundation ✔ Walls ✔ Doors ✔ Windows ✔ Rooms ✔ Exactly the same way HTML creates the structure of a website.

Basic HTML Structure


<!DOCTYPE html>
<html>

<head>
    <title>My Website</title>
</head>

<body>

    <h1>Hello World</h1>

</body>

</html>

Important Points

  • HTML creates website structure.
  • HTML uses Tags.
  • HTML is easy to learn.
  • Every website starts with HTML.
  • HTML works together with CSS and JavaScript.
